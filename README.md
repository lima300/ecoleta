### Configurar API

```bash
# Entre na pasta da API
$ cd ecoleta/server

# Instale as dependências
$ yarn install

# Rode as migrations
$ yarn knex:migrate

# Rode os seeds
$ yarn knex:seed

# Inicie a API
$ yarn dev

```

### Iniciar front-end web

```bash
# Entre na pasta web
$ cd ecoleta/web

# Instale as dependências
$ yarn install

# Inicie o servidor
$ yarn start

```

### Iniciar mobile expo

```bash
# Entre na pasta mobile
$ cd ecoleta/mobile

# Instale as dependências
$ yarn install

# Inicie o servidor
$ yarn start

# Expo irá abrir, você então deve escanear o QR Code com a câmera do celular.

```
